Nginx静态资源路径配置

文/年容

踩坑日记，简单记录。

server {
    listen       8088;
    server_name  localhost;

    #第1处
    location /html {
      root   /Users/pengnr/Desktop/;
      index  index.html index.htm;
    }

    #第2处
    location /html {
      alias  /Users/pengnr/Desktop/html/;
      index  index.html index.htm;
    }
}

静态资源的绝对路径相同，访问http://localhost:8088/html，找到静态资源的方式如下：

第1处

/Users/pengnr/Desktop/ + html >>> /Users/pengnr/Desktop/html

第2处

/html >>> /Users/pengnr/Desktop/html/

注意第2处alias结尾有反斜杠，第1处root结尾反斜杠可有可无
